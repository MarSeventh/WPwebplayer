🎵 WP Music Player - 使用指南

📦 发布准备：将 CSS 和 JS 文件上传到你的 CDN 或静态资源服务器（如 npm.webcache.cn、jsDelivr、unpkg 等）

📁 文件结构

  • wp-music-player.css - 播放器样式文件
  • wp-music-player.js - 播放器功能文件

🚀 使用方法

方法一：自定义元素（推荐）

最简单的使用方式，类似你现有的 meting-js 方案：

<!-- 引入样式和脚本 --> <link rel="stylesheet" href="https://your-cdn.com/wp-music-player.css"> <script src="https://your-cdn.com/wp-music-player.js"></script> <!-- 播放器标签 --> <wp-music-player src="/wp-content/uploads/music/my-song.mp3" title="我的音乐" artist="艺术家名称" loop="true" autoplay="false" volume="0.5" fixed="true" mini="true"> </wp-music-player>

方法二：JavaScript 配置

通过 JavaScript 配置播放器：

<!-- 引入样式和脚本 --> <link rel="stylesheet" href="https://your-cdn.com/wp-music-player.css"> <script src="https://your-cdn.com/wp-music-player.js"></script> <!-- 配置脚本 --> <script data-wp-music-config> { "src": "/wp-content/uploads/music/my-song.mp3", "title": "我的音乐", "artist": "艺术家名称", "loop": true, "autoplay": false, "volume": 0.5, "fixed": true, "mini": true } </script>

方法三：手动初始化

完全手动控制播放器：

<!-- 引入样式和脚本 --> <link rel="stylesheet" href="https://your-cdn.com/wp-music-player.css"> <script src="https://your-cdn.com/wp-music-player.js"></script> <script> document.addEventListener('DOMContentLoaded', function() { const player = new WPMusicPlayer({ src: '/wp-content/uploads/music/my-song.mp3', title: '我的音乐', artist: '艺术家名称', loop: true, autoplay: false, volume: 0.5, fixed: true, mini: true }); }); </script>
